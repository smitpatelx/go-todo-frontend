# Mufflr
- Dependency: `React, Nextjs, TailwindCss, Formik`


## Things to de before starting
- Install VS-CODE plugins for the project
  + EditorConfig
  + Tailwind CSS IntelliSense


## Start
- Clone the project
  + `git clone [url]`
- `yarn` (Install node modules)
- `yarn dev` (Spin up development server)
